BGP Network protocol Project



This Project has two networks. The "inner" network made up of a Windows 10 PC that is connected to three routers configured to run BGP and has port fowarding enabled. The "outer" network is a Ubuntu PC with a different IP address associated with the "outer" network that is connected to the OpensenseDev Router.
